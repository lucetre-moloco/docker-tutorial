### running docker image
* `docker pull redis`
* `docker images`
* `docker run redis`
* `docker ps`
* detach mode `docker run -d redis`
* `docker stop <container-id>`
* `docker start <container-id>`
* `docker ps -a`
* `docker run redis:4.0`
